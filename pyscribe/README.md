# PyScribe

PyScribe is a Python script for generating piano fingerings for musicxml files.

## Setup

Install music21 and deap

```bash
pip install music21
pip install deap
```

## Usage

Run the script on a musicxml file

```bash
usage: pyscribe.py [-h] filename
```
